## How It Works

The counter value is stored in JavaScript and updated through button interactions.  
Each change updates the display to simulate a mechanical counting motion rather than a plain number update.

## Getting Started

### Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Kingkhansk/Mechanical-Counter.git
   ```

2. Open the project folder:
   ```bash
   cd Mechanical-Counter
   ```

3. Open `index.html` in any modern web browser.
